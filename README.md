# 🌌 Cyber-Isometric Dashboard 3D

Un tableau de bord technologique et futuriste avec un effet de perspective isométrique tridimensionnelle, conçu entièrement en technologies web fondamentales (**HTML5, CSS3 et JavaScript pur**).

Ce projet simule une interface de programmation dynamique en mouvement, inspirée des univers de science-fiction et du design cyberpunk.

---

## 🚀 Fonctionnalités

* **Perspective 3D Isométrique :** Utilisation avancée des propriétés CSS `perspective` et `transform3d` pour coucher les éléments dans l'espace.
* **Animation de Flottaison (Lévitation) :** Le panneau principal oscille doucement dans les airs pour un effet high-tech immersif.
* **Flux de Code Dynamique :** Injection continue de lignes de code asynchrones en JavaScript imitant un terminal actif en temps réel.
* **Design Fluide & Lumineux :** Esthétique en mode sombre avec des accents néons (cyan et bleu) et des ombres portées réactives (`box-shadow`).

---

## 🛠️ Technologies Utilisées

Pour ce projet de formation, l'accent a été mis sur la maîtrise du code pur, sans framework externe :

* **HTML5 :** Structuration sémantique de la scène et des panneaux de contrôle.
* **CSS3 :** Gestion de la 3D, de la grille technologique au sol (`linear-gradient`), des animations clés (`@keyframes`), et du masquage des débordements (`overflow`).
* **JavaScript (Vanilla) :** Moteur de génération aléatoire de code et gestion du défilement automatique du terminal.
